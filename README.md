# Predicting Job Change Intentions Using Machine Learning

## Project Overview

This project focuses on predicting whether data science candidates are likely to change jobs using machine learning techniques and HR analytics data. The goal is to support data-driven decision-making in human resources by identifying patterns related to employee job-change intentions.

The project includes data preprocessing, exploratory data analysis (EDA), visualization, feature engineering, and the implementation of machine learning classification models.

---

## Objectives

* Analyze HR analytics data related to candidate demographics, education, experience, and employment background.
* Identify factors influencing job change intentions.
* Build and evaluate machine learning models for binary classification.
* Compare model performance using evaluation metrics such as accuracy, precision, confusion matrix, and classification reports.

---

## Dataset

The project uses the **HR Analytics: Job Change of Data Scientists** dataset containing information about:

* Education level
* Relevant experience
* Company type
* Training hours
* City development index
* Employment background
* Target variable indicating job change intention

## Machine Learning Models

The following models were implemented and evaluated:

* Logistic Regression
* XGBoost Classifier

The XGBoost model achieved the best overall performance with approximately 85% accuracy.

## Results

The project demonstrates that machine learning models, particularly XGBoost, can effectively predict job change intentions based on HR-related features. The findings highlight the potential of predictive analytics in supporting recruitment and employee retention strategies.
